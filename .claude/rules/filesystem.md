# Filesystem Rules

## File Paths

Always use paths relative to the project root. Never use absolute paths.  
Using the `./` prefix is preferred for clarity.

- ✅ Correct: `./directives/architecture.md`
- ❌ Incorrect: `C:/Users/renat/OneDrive/xp/TI2_Sistemas/Scraping2/directives/architecture.md`

---

## Temporary Files

All temporary files generated by AI Agents during execution — including debug logs, HTML dumps, screenshots, diagnostic outputs, selector tests, and any other throwaway artifacts — **MUST be saved exclusively in `/tempFiles`**.
